# Café Management System

A Java-based desktop application designed to replace manual café management processes with an automated, structured system.

## Project Description

**Café Management System** is a lightweight Java application that follows the **Model-View-Controller (MVC)** architectural pattern. The goal of the project is to provide an efficient digital solution for managing daily café operations, including menu handling, order processing, inventory tracking, and basic reporting — effectively "replacing the manager" for routine tasks.

The application is built using pure Java (no external frameworks or dependencies at this stage), making it easy to compile and run on any system with a Java Development Kit (JDK) installed.

## Project Structure

The repository is organized according to the MVC design pattern:
CafeManagment-System/
├── controller/          # Contains controller classes (business logic and user input handling)
├── model/               # Contains entity/model classes (data structures and business objects)
├── view/                # Contains view classes (user interface components)
├── Main.java            # Application entry point
└── Main.class           # Compiled bytecode (should be excluded from version control)
